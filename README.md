#  Gerenciador de Turma Acadêmica 
    Projeto desenvolvido para a disciplina de Estrutura de Dados no curso de Tecnologia/Engenharia da UDF (Centro Universitário do Distrito Federal).

    Este sistema em C++ foi projetado para facilitar o controle de notas, frequências e geração de relatórios de desempenho acadêmico. O foco principal é a implementação eficiente de estruturas de dados para manipulação de registros de alunos.

##  Funcionalidades
    Cadastro de Alunos: Inserção de nome, matrícula e notas.

    Cálculo de Médias: Processamento automático de médias aritméticas ou ponderadas.

    Relatórios de Desempenho: Listagem de alunos aprovados, em exame e reprovados.

    Ordenação: Exibição da turma por ordem alfabética ou por classificação de notas.

    Busca Eficiente: Localização de registros por número de matrícula.

##  Tecnologias e Conceitos Aplicados
    Linguagem: C++

    Estruturas Utilizadas: [Ex: Listas Encadeadas / Árvores Binárias / Vetores Dinâmicos]

    Algoritmos: Ordenação (Bubble Sort/Quick Sort) e Busca Binária.

    Paradigma: Programação Estruturada ou Orientada a Objetos.
